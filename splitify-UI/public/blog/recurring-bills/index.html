How to Split Recurring Bills

Rent, utilities, and subscriptions without spreadsheets

By Splitify Team·January 8, 2025·7 min read

Recurring costs are predictable—your process should be too. Automate rent, utilities, and shared subscriptions with recurring requests that go out on your schedule, with your preferred split.

Create a recurring request

  1. Choose frequency (weekly, monthly) and start date.
  2. Set a due date offset (e.g., due 3 days after send).
  3. Select the split method and add a short note.

Adjust for changes

Someone moving out or pausing for a month? Edit participants and amounts once; changes apply to future cycles. You can also skip a single cycle without breaking the schedule.

Stay on top of payments

Monitor status from the dashboard and let smart reminders do the nudging—so you don't have to. Reminders pause automatically once someone pays.

Pro tips

  • Send a few days before the bill is due to give people time.
  • Include context in your note (e.g., "electric + internet").
  • Use percentages when consumption isn't equal.

Set it once, then stop stressing. Create a recurring request and let Splitify keep it on track.
